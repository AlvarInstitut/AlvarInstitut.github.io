7.- Designer GUI

Una vegada hem fet un repàs rapidet pels components Swing, és hora de veure un entorn més agradable per anar col·locant els distints components. La funcionalitat dels components és la mateixa, ja que el WindowBuilder només fa que col·locar els components Swing.

En Eclipse hi ha un plugin anomenat WindowBuilder per a poder col·locar els components gràfics de Swing de forma visual i còmoda.

En IntelliJ IDEA encara és més fàcil perquè tenim una eina ja integrada, Designer GUI.

Quan dissenyem una finestra amb diferents components, Designer GUI crearà una classe java i un formulari associat a aquesta classe java, que serà realment un fitxer XML amb la definició de tots els components (per tant molt similar a l'estructura en Android). Segurament el més còmode seria reballar en Java, ja que la classe és java, però podrem fer perfectament el programa principal en Kotlin i fer referència a la classe Java.
