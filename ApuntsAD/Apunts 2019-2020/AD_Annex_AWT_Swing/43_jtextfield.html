4.3.- JTextField

Es el component corresponent a un quadre de text. Les seues propietats i mètodes bàsics són:

  • Constructor: en el moment de construir li podem dir opcionalment la grandària (en caràcters)

    JTextField c1 = new JTextField(25);

  • getText(): torna el contingut del JTextField
  • setText(String): modifica el contingut del JTextField.
  • setEditable(boolean): marcarà si el es pot conviar el seu contingut o no.

El següent exemple trasllada el contingut del primer JTextField al segon quan s'apreta el botó:

import java.awt.FlowLayout;
import java.awt.GridLayout;
import java.awt.event.ActionEvent;
import java.awt.event.ActionListener;

import javax.swing.JButton;
import javax.swing.JFrame;
import javax.swing.JPanel;
import javax.swing.JTextField;

public class Finestra extends JFrame implements ActionListener{
    JTextField t1 = new JTextField(15);
    JTextField t2 = new JTextField(15);
    JButton boto1 = new JButton(">>");
    JButton eixir = new JButton("Eixir");
    
    public void iniciar(){
        setSize(500,100);
        getContentPane().setLayout(new GridLayout(2,1));
        JPanel panell1 = new JPanel(new FlowLayout());
        JPanel panell2 = new JPanel(new FlowLayout());
        
        getContentPane().add(panell1);
        panell1.add(t1);
        panell1.add(boto1);
        panell1.add(t2);
        getContentPane().add(panell2);
        panell2.add(eixir);
        setVisible(true);
        
        boto1.addActionListener(this);
        eixir.addActionListener(this);
    }

    @Override
    public void actionPerformed(ActionEvent e) {
        if (e.getSource()==boto1){
            t2.setText(t1.getText());
        }
        if (e.getSource()==eixir){
            System.exit(0);
        }
    }
}
